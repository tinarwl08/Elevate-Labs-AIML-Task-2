Boston Housing Dataset — Exploratory Data Analysis (EDA)
Project Overview:
This project performs Exploratory Data Analysis (EDA) on the Boston Housing dataset to
understand patterns and relationships affecting house prices.
Objectives:
- Understand dataset structure
- Generate summary statistics
- Visualize distributions and relationships
- Identify patterns and anomalies
Dataset:
- 506 rows, 14 columns
- Target: medv (house price)
Tools Used:
- Python, Pandas, Matplotlib, Seaborn, Plotly
Key Insights:
- More rooms (rm) → higher prices
- Higher lstat → lower prices
- Outliers present in crim, tax
- Strong correlations exist
Conclusion:
EDA helps understand data and prepare for machine learning.
